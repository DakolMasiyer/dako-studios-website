## Design Notes
- `text-[40vw]` watermark uses `text-foreground/[0.04]` — adapts to both light and dark mode automatically via existing CSS tokens
- `DakoLogo` is a client component but can be imported into a server component — Next.js handles the boundary
- No framer-motion, no new dependencies, no new files
- `font-display` maps to Space Grotesk via the existing Tailwind theme config
