## How to Use This Guide

Every objection listed here maps to a content type, an outreach message, or a proposal conversation response. Before publishing any post or sending any outreach, check it against the relevant objections. The best content pre-empts objections before the prospect voices them.

This version covers single-sided client objections only — there is no separate supply-side objection set, since Dako Studios sells directly to clients rather than running a two-sided marketplace.

## Objection Stress Test Checklist

Before publishing any content piece or outreach message, run it through this checklist:

- [ ] Does it pre-empt at least one objection without being defensive?
- [ ] Does it lead with proof (outcome, turnaround, named work) not a claim?
- [ ] Does it address the *real concern* underneath the objection, not the surface one?
- [ ] Is the CTA smaller than the objection expects?
- [ ] Does it avoid abstract language ("transforming", "revolutionising", "changing the game")?
